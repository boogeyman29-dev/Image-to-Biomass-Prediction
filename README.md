Image to Biomass Prediction using Computer Vision

Project Overview

This project predicts plant biomass using image processing techniques. It analyzes green vegetation in images and estimates biomass using a machine learning model.

Features

* Upload plant images
* Extract green pixel ratio using OpenCV
* Predict biomass using Linear Regression
* Interactive UI using Gradio

🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-learn
* Gradio

How to Run

1. Open the notebook in Google Colab
2. Upload plant images
3. Run all cells
4. Get biomass predictions

Output

* Green ratio detection
* Biomass prediction values

Future Improvements

* Implement CNN (Deep Learning)
